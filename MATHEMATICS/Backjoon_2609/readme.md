# [[2609]최대 공약수와 최소 공배수](https://www.acmicpc.net/problem/2609)

## 수학을 이용한 풀이

- 최소 공약수 구하기

    - 두 수 중 작은 수 a, 큰 수 b
    - b를 a로 나눈다.
    - 나눠진 값이 0이라면 b는 a로 나눠지기 때문에 a가 최대 공약수가 된다.
    - 나눠진 값이 0이 아니라면 큰 수 b는 작은 수 a로, 작은 수 a는 b/a인 값을 넣어 위 과정을 반복한다.
  
- 최대 공배수 구하기

    - 두 수의 최대 공약수를 유클리드 호제법을 통해 구한다.
    - 두 수 a, b를 곱한 뒤, 최대 공약수로 나눈 값을 최소 공배수로 출력한다.
