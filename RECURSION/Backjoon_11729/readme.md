# [[11729]하노이 탑 이동 순서](https://www.acmicpc.net/problem/11729)

## Recursion을 이용한 풀이

- n = 1일 때 최소 이동 1
- n = 2일 때 최소 이동 3
- n = 3일 때 최소 이동 7
- ...
- n 일 때 최소 이동 `2^n - 1`
