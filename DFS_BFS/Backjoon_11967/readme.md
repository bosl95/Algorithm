# [[11967]불켜기](https://www.acmicpc.net/problem/11967)

## BFS/DFS를 이용한 풀이

- 베시는 (0, 0)에서 출발한다.
- 베시는 상, 하, 좌, 우로 이동한다.

- 첫번째 방을 탐색 큐에 넣고 시작한다.
- 탐색 큐를 실행한다.
- 현재 위치에서 켤 수 있는 불을 킨다. (카운트 증가)
  - 이때 이미 켜져있는 방인 경우 continue
  - 켜져있지 않다면 불을 킨다.
  - 불을 켤 수 있는 방이라면 해당 위치의 방에서 상,하,좌,우를 탐색 ==> 방문했던 적있는 방이 있다면 큐에 추가한다.
    
- 큐가 비어있을 때까지 탐색을 반복한다.
